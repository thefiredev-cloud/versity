# Versity

> Education That Adapts

Complete adaptive learning paths, assessments, and training — built by AI, personalized for every learner.

## Built for Claude Mythos

This landing page was built as part of The Fire Dev's moonshot product suite.

## Tech Stack

- Next.js 14 (App Router, static export)
- React 18
- Three.js via @react-three/fiber + @react-three/drei
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Outputs to `out/` directory (static export, ready for Netlify/Vercel).

## Features

- **Adaptive Paths** — Learning paths that evolve with every response
- **Auto-Assessment** — Continuously generated assessments measuring real understanding
- **Skills Mapping** — Visual skills graph showing mastery and gaps
- **Progress Intelligence** — Predictive analytics on learning trajectories
